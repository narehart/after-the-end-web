motivators:

- character has status meters
  - meters deplete under various conditions
    - time
    - injury
    - status
    - resources
    - combat
    - encounter
    - weather
  - meters increase under various conditions
    - time
    - status
    - resources
    - encounter
    - weather

actions:

- character can increase meters through actions
  - actions don't always guarantee increase, there is risk/reward
  - actions progress time
  - available actions
    - rest
      - increases some meters, decreases others
    - explore
      - reveals additional locations, resources
      - can lead to combat, encounters
    - scavange
      - gathers resources (consumable, craftable)

time:

- decreses with each action point used

injury:

- sustained during exploration, combat

resources:

- found scavanging or at the end of encounters

combat:

- happens while exploring, encounters, or resting

encounter:

- happens when entering a location

weather:

- changes over time
